kernel of a homomorphism between algebraic systems


Let f:(A,O)→(B,O) be a homomorphism between two algebraic systems A and B (with O as the operator set). Each element b∈B corresponds to a subset K⁢(b):=f-1⁢(b) in A. Then {K⁢(b)∣b∈B} forms a partition of A. The kernel ker⁡(f) of f is defined to be

ker⁡(f):=⋃b∈BK⁢(b)×K⁢(b).

It is easy to see that ker⁡(f)={(x,y)∈A×A∣f⁢(x)=f⁢(y)}. Since it is a subset of A×A, it is relation on A. Furthermore, it is an equivalence relation on A: 11In general, {Ni} is a partition of a set A iff ⋃Ni2 is an equivalence relation on A.

  1. 1.

    ker⁡(f) is reflexive: for any a∈A, a∈K⁢(f⁢(a)), so that (a,a)∈K⁢(f⁢(a))2⊆ker⁡(f)

  2. 2.

    ker⁡(f) is symmetric: if (a1,a2)∈ker⁡(f), then f⁢(a1)=f⁢(a2), so that (a2,a1)∈ker⁡(f)

  3. 3.

    ker⁡(f) is transitive: if (a1,a2),(a2,a3)∈ker⁡(f), then f⁢(a1)=f⁢(a2)=f⁢(a3), so (a1,a3)∈ker⁡(f).

We write a1≡a2(modker⁡(f)) to denote (a1,a2)∈ker⁡(f).

In fact, ker⁡(f) is a congruence relation: for any n-ary operator symbol ω∈O, suppose c1,…,cn and d1,…,dn are two sets of elements in A with ci≡dimodker⁡(f). Then

f(ωA(c1,…,cn)=ωB(f(c1),…,f(cn))=ωB(f(d1),…,f(dn))=f(ωA(d1,…,dn)),

so ωA⁢(c1,…,cn)≡ωA⁢(d1,…,dn)(modker⁡(f)). For this reason, ker⁡(f) is also called the congruence induced by f.

Example. If A,B are groups and f:A→B is a group homomorphism. Then the kernel of f, using the definition above is just the union of the square of the cosets of

N={x∣f⁢(x)=e},

the traditional definition of the kernel of a group homomorphism (where e is the identity of B).

Remark. The above can be generalized. See the analog (http://planetmath.org/KernelOfAHomomorphismIsACongruence) in model theory.
